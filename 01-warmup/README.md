Create a professional visualization of how LLMs are trained or a similar topic that interests you.

It should include the pre-training, mid-training and post-training stage and show what kind of data and losses are applied in each step, or similar level of detail for the topic you chose.

First, draw a sketch of your preferred visualization on paper.
Then use the agent to create a professional digitalized version of it.

### Coding Agents 101

If you have not done so yet, download and install Codex from this link: https://developers.openai.com/codex/cli
Open a terminal on your computer to start the agent like this:
```
codex
```

Use the provided API-key to authenticate.
Feel free to use the Codex App instead (only available on Mac).
I recommend using the default settings (latest model, medium reasoning).
